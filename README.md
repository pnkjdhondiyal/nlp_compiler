# nlp_compiler
An intelligent compiler that enables natural language support for programming by converting human-like instructions into executable code. Built using Python, NLP (spaCy, GPT), and LLVM, this project aims to bridge the gap between human communication and machine code execution.

🚀 Features
🧠 Natural Language Processing (NLP): Converts natural language instructions into structured programming constructs.

🔤 Multi-Language Support: Supports multiple programming languages with intelligent translation.

🎯 LLVM Backend: Generates optimized machine code using LLVM for better performance.

🔍 Syntax & Semantic Analysis: Uses NLP techniques to understand and validate user instructions.

⚡ Interactive Code Generation: Dynamically translates human-like queries into executable code snippets.

🛠️ Technologies Used
Python 🐍 (Core programming language)

spaCy 🌍 (For NLP processing and entity recognition)

GPT Models 🤖 (For advanced text interpretation)

LLVM 🏗️ (For generating optimized machine code)

📌 How It Works
Accepts human-readable programming instructions (e.g., "Create a loop that runs 10 times").

Uses NLP to parse and understand the intent.

Translates the instruction into a programming language syntax.

Compiles the generated code using LLVM for execution
